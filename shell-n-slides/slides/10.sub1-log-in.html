Log into the web interface

Since we have crashed the web server on the camera with out exploit,
in order to log into the camera we need to reset it.

  • Click the reset camera button in the top right
  • Open a new tab in the browser and enter the IP port in the format http://[TARGET IP]:8080
  • Use the password you found to log in!

There is no image as the camera is emulated and has no sensor. Go to the next page to reset this lab for the next user.
