Nmap

nmap is a network scanning tool that can be used to discover hosts and services on a computer network.

It is used to discover hosts and services on a computer network, thus creating a "map" of the network.

Let's try a basic scan.

Run this command in the terminal:

sudo nmap --disable-arp-ping -PE -sn

This will skip arp and do a ping scan on the network without any port scanning.
